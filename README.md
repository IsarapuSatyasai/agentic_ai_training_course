# Applied Generative AI and Agentic Systems

## 🚀 Project Overview
The goal of this project is to transition from foundational programming to architecting production-ready **Agentic AI systems**. This repository documents the development of autonomous systems built with **Python**, **LangChain**, and **LangGraph** that can retrieve knowledge, reason through tasks, and interact with users.

## 🛠️ Tools & Technologies
* **Languages:** Python (Beginner to AI-specific Advanced)
* **Orchestration:** LangChain, LangGraph
* **Databases:** FAISS, ChromaDB
* **Deployment:** FastAPI, Streamlit, Docker, AWS
* **Collaboration:** GitHub, Google Meet, Google Drive

---

## 📈 8-Week Roadmap & Milestones

### Milestone 1: Technical & AI Foundations
* **Week 1: Programming & ML Quick-Start**
    * Intensive focus on Python syntax, data structures, and functions for AI.
    * Practical exploration of Machine Learning vs. Deep Learning foundations.
    * **Output:** CLI AI Assistant with conversation history.
* **Week 2: Prompt Engineering**
    * Implementation of Zero-shot, Few-shot, and Chain-of-Thought reasoning.
    * **Output:** AI Blog Generator with structured JSON outputs.

### Milestone 2: Retrieval & Knowledge Management
* **Week 3: LangChain & Vector Search**
    * Document indexing and similarity search with FAISS and ChromaDB.
    * **Output:** Semantic Search Engine.
* **Week 4: Advanced RAG Architecture**
    * Optimization via advanced chunking and metadata filtering.
    * **Output:** PDF Question Answering System.

### Milestone 3: Agentic Reasoning & Collaboration
* **Week 5: AI Agents & Tool Usage**
    * Building reasoning loops, tool usage, and agent memory.
    * **Output:** AI Research Agent for autonomous info retrieval.
* **Week 6: Multi-Agent Systems (LangGraph)**
    * Orchestrating Planner, Researcher, and Writer agents using graph-based workflows.
    * **Output:** Multi-Agent Research System.

### Milestone 4: Safety, Regulation & Production
* **Week 7: Evaluation, NSWE & Protection**
    * Testing agent trajectory and response reliability.
    * Implementing NSWE (Work Health and Safety) standards and guardrails against prompt injection.
    * **Output:** Detailed Evaluation & Safety Documentation.
* **Week 8: Final Deployment & Capstone**
    * Backend development with FastAPI and containerization with Docker.
    * **Capstone Project:** Agentic AI Medical & Knowledge Assistant.

---

## 🎯 Key Outcomes
* **Programming Foundations:** Mastery of data preprocessing and the transition to Deep Learning.
* **RAG Pipelines:** Expertise in document indexing and retrieval using vector databases.
* **Autonomous Agents:** Design of multi-agent workflows capable of planning and tool execution.
* **Safety & Regulation:** Implementation of evaluation frameworks and ethical protection standards.
* **Production Deployment:** Preparation of scalable AI applications for AWS hosting.
